asdfghjkl.psd : Before Photoshop era

This is the project I did in a film photography course during my first semester at California College of the Arts. It was my first time ever using a film camera, developing films and printing in the darkroom. I decided to manipulate these photos in the dark room to make them look as if they were done in photoshop. After hours and hours spent in the darkroom trying to get everything right, I'm very much happy with the results.

x

A book about design a book about a block

The project we were challenged to do was called "a book about a block". We got to pick one block in San Francisco and create a book out of it. We went through numbers of processes to create the contents. Halfway through the semester, I realized how interesting these processes were and wanted to document them so I made a quick turn and here it is: a book about designing a book about a block.

I discovered my love for metadata, bold design and #FFC42C while working on this project and I'm glad that I took a full advantage of being in school by blending the assignment and turning it into my own.

Pong CSS typeface

This project is to create a typeface using only HTML and CSS with a minimum of 2 visual states and a website to display a complete typeface in context. So I designed this typeface that only consists of squares and rectangles to keep the minimal aesthetic and similar to this old computer game "Pong".
